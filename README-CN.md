Gateway 是一个高性能的、各类设备插件的集中通讯调度模块，把各类异构数据统一转换为标准模型，完成实时数据的存储、报警、联动、任务规划等核心功能。

### 平台支持
Gateway 当前支持 Windows、Linux、MacOS平台。
